About

Counting the number of occurrences of every substring of length k (so-called k-mer) in a given string S is an important procedure in bioinformatics.

Problem

Counting k-mers is challenging, because it should be both
fast and performed using as little memory as possible. A naïve
approach is to use a standard hash table, with k-mers as keys
and their counts as values. This solution is both memory
consuming and hard to parallelize effectively
